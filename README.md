## Nombre del autor

## Descripción de la práctica
Este proyecto consiste en la creación de un Moodboard Personal interactivo utilizando tecnologías web modernas. El objetivo principal es demostrar el dominio de la estructura semántica en HTML5 y la maquetación avanzada mediante CSS Grid Layout.

El diseño refleja mi identidad como estudiante de programación y mis intereses personales (pádel y naturaleza), manteniendo una estructura rígida de rejilla que se adapta dinámicamente según el dispositivo desde el que se visualice.

## Tecnologías utilizadas
Para el desarrollo de este proyecto se han empleado las siguientes tecnologías y metodologías:

-HTML5 Semántico: Uso de etiquetas estructurales (<header>, <main>, <article>, <section>, <footer>) para garantizar la accesibilidad y validación W3C.

-CSS3 Avanzado:

    -CSS Grid: Implementación de grid-template-areas para la distribución de los componentes.

    -Variables CSS: Uso de :root para gestionar de forma centralizada la paleta de colores y bordes.

    -CSS Nesting: Aplicado en la sección de proyectos para mejorar la legibilidad del código.

    -Flexbox: Utilizado para la alineación interna de elementos dentro de las tarjetas.

    -Diseño Responsive: Implementación de Media Queries para adaptar el diseño a resoluciones de Tablet y Móvil.

    -Interactividad: Efectos hover con transiciones suaves para mejorar la experiencia de usuario.
